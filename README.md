# GoTravel Project
Welcome to my GoTravel repository! This is a project I developed as a first-year student to demonstrate my foundational skills in Front-End Development. The goal was to build a fully functional, responsive website for a travel agency, incorporating user interaction, dynamic filtering, and interactive components.

## 📚 What I learned & practiced
Building this project helped me gain a solid understanding of how to structure and style web pages. Here are the core areas I focused on:

Responsive Web Design: Mastering CSS Flexbox and Grid, along with Bootstrap’s layout system, to ensure the site looks good on all devices.

Component Integration: Integrating Bootstrap to handle complex UI elements like carousels and responsive grids.

JavaScript Logic: Implementing dynamic features like filtering, price calculation, and form validation using Vanilla JS.

Form Management: Creating secure and user-friendly registration, login, and booking forms.

Version Control: Learning to manage my project effectively using Git and GitHub.

## 🛠 Technologies Used
HTML5

CSS3 (Custom styles & Media Queries)

Bootstrap 5 (Integrated for UI components and responsive layouts)

JavaScript (ES6+)
